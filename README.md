# 🏋️ FitNation – Projet Machine Learning

Projet réalisé dans le cadre de ma **formation Data Analyst chez DataSuits**.

---

## 🧠 C'est quoi ce projet ?

FitNation est une chaîne de clubs de sport fictive. À partir de données clients, j'ai construit deux modèles de machine learning :

- **Modèle 1** : prédire le revenu mensuel d'un client
- **Modèle 2** : prédire si un client va résilier son abonnement (churn)

---

## 📁 Fichiers du projet

| Fichier | Description |
|--------|-------------|

| `pred_clients_2024.xls` | Résultats des prédictions sur les clients 2024 |
